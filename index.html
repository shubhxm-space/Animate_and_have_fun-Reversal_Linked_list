Algorithm Steps

  1. Initialize prev = NULL, curr = head
  2. Save next = curr.next
  3. Reverse link: curr.next = prev
  4. Move prev = curr
  5. Move curr = next
  6. Repeat until curr == NULL
  7. Set head = prev (new head)
